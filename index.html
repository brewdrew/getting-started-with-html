Major pieces of the web: Browser -> Internet -> HTTP Request to Server -> and same in reverse

HTML - Hypertext markup language, main web doc type
  • Heart of the web
  • Made up of text content - what you see
  • Markup - what it looks like
  • References to - images, videos etc
  • Other docs
  • Links

HTML Markup - made up of tags such as b for bold, em for emphasis or italics

Image tags (img) do not include a closing tag, these are called void tags

HTML elements are either inline, or block. Block elements form an invisible box around the content inside them. Examples of inline elements are b, img or span. P, div and others are examples of block elements.

HTML attributes are part of HTML elements, for example using the href attribute to provide the URL in an anchor tag.
